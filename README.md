# TryHackMe - Kenobi Writeup

## Overview

This repository contains a step-by-step writeup for the Kenobi room.

The goal of this lab is to practice:

* Enumeration
* Exploitation
* Linux Privilege Escalation

## Attack Path

1. Network scanning
2. SMB enumeration
3. FTP enumeration
4. Exploiting ProFTPD mod_copy
5. NFS mounting
6. SSH key abuse
7. Privilege escalation using PATH hijacking

## Repository Structure

* `01-recon` → Initial scans
* `02-enumeration` → Service enumeration
* `03-exploitation` → Exploiting vulnerabilities
* `04-initial-access` → Getting shell
* `05-privilege-escalation` → Root access

## Skills Learned

* Nmap scanning
* SMB enumeration
* FTP exploitation
* NFS mounts
* SSH key authentication
* Linux privilege escalation
* PATH hijacking
